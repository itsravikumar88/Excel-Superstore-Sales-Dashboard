# Excel Sales Dashboard

## Project Overview

This project demonstrates the creation of an interactive Sales Dashboard in Microsoft Excel to analyze sales performance, profit trends, customer behavior, and regional performance.

The dashboard helps stakeholders make data-driven decisions by providing clear visual insights into business performance.

---

## Business Problem

Management wants to monitor:

* Total Sales
* Total Profit
* Top Performing Categories
* Regional Performance
* Monthly Sales Trends
* Customer Segments

The dashboard provides a centralized view of key business metrics.

---

## Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* Conditional Formatting
* Data Cleaning
* KPI Cards

---

## Dashboard Features

### KPI Metrics

* Total Sales
* Total Profit
* Total Orders
* Average Order Value

### Visualizations

* Monthly Sales Trend
* Profit by Category
* Sales by Region
* Top 10 Products
* Customer Segment Analysis

### Filters

* Region
* Category
* Segment
* Year

---

## Key Insights

* Technology category generated the highest revenue.
* West region contributed the highest sales.
* Profitability increased significantly during Q4.
* Consumer segment accounted for the majority of sales.
